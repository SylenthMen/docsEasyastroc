# Componente Button

El componente `Button` es un botón reutilizable que permite personalizar el texto y el color del texto. Puede ser utilizado dentro de un enlace (`<a>`) o de forma independiente.

## Props

- `nameButton` (string): El texto que aparecerá dentro del botón.
- `colorText` (string): Clases CSS para estilizar el color del texto del botón.

## Ejemplo de uso

A continuación, un ejemplo de cómo utilizar el componente `Button` dentro de un enlace y de forma independiente:

```astro
<a href="enlace">
  <Button nameButton="continuar" colorText="text-[#05ce91]" />
</a>

<Button nameButton="continuar" colorText="text-[#05ce91]" />
```
