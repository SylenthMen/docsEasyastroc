## Button

El componente `Button` es un botón interactivo que permite a los usuarios realizar acciones en la página. Los estilos del botón, como los colores de texto, fondo y borde, se pueden personalizar a través de las props, además de definir el comportamiento cuando el botón es clicado.

### Props

| Propiedad        | Tipo     | Descripción                                                 |
|------------------|----------|-------------------------------------------------------------|
| `nameButton`     | `string` | Texto que aparecerá en el botón.                            |
| `colorText`      | `string` | Clases de Tailwind para personalizar el color del texto.     |
| `colorBackground`| `string` | Clases de Tailwind para personalizar el color de fondo.      |
| `colorBorder`    | `string` | Clases de Tailwind para personalizar el color del borde.     |
| `onClick`        | `string` | Función a ejecutar cuando se hace clic en el botón.          |

### Ejemplo de uso

```astro
<Button
  nameButton="Click me"
  colorText="text-white"
  colorBackground="bg-green-600"
  colorBorder="border-blue-600"
  onClick="console.log('Button clicked!')"
  styleButton="p-2"
/>
```
