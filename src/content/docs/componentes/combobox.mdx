# Componente ComboBox

El componente `ComboBox` permite al usuario seleccionar una opción de una lista desplegable con estilos personalizados. Es ideal para formularios o situaciones donde se necesita seleccionar un valor entre varias opciones.

## Props

- `titleComboBox` (string): El título que describe el propósito del ComboBox.
- `textComboBox` (string): El texto que aparece como placeholder en el campo de entrada del ComboBox.
- `styleCombo` (string): Clases CSS para personalizar el estilo del contenedor del ComboBox.
- `styleInput` (string): Clases CSS para estilizar el campo de entrada del ComboBox.
- `styleBoxOptions` (string): Clases CSS para estilizar el contenedor de las opciones desplegables.
- `styleOptions` (string): Clases CSS para estilizar cada opción individual dentro del ComboBox.
- `widthCombo` (string): Clases CSS para definir el ancho del ComboBox.

## Ejemplo de uso

A continuación, un ejemplo de cómo utilizar el componente `ComboBox`:

```astro
<ComboBox
  titleComboBox="Elige tu lenguaje favorito:"
  textComboBox="Selecciona un lenguaje de programación"
  styleCombo="bg-neutral-950 text-white"
  styleInput="border-b border-neutral-800 bg-neutral-800 text-purple-500 focus:outline-none"
  styleBoxOptions="bg-neutral-950 text-white"
  styleOptions="text-[#05ce91] hover:bg-[#002f24] cursor-pointer"
  widthCombo="w-96"
/>
```
