## FormularioContacto

El componente `FormularioContacto` permite crear formularios de contacto personalizables, con un título, estilos y campos de entrada. Incluye un botón de envío que puede ser personalizado y gestionado con eventos.

### Props

| Propiedad         | Tipo      | Descripción                                                                                                                                           |
|-------------------|-----------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| `titleStyle`      | `string`  | Clases CSS para estilizar el título del formulario, como colores, tamaño de fuente y alineación.                                                        |
| `titleFormulario` | `string`  | El título que aparecerá en la parte superior del formulario, generalmente indicando su propósito.                                                       |
| `styleFormulario` | `string`  | Clases de Tailwind para estilizar el contenedor del formulario, como colores de fondo, bordes y efectos visuales (ej: `backdrop-blur`).                  |
| `colorLabel`      | `string`  | Clases de Tailwind para definir el color de las etiquetas de los campos de entrada del formulario.                                                      |
| `styleInput`      | `string`  | Clases de Tailwind CSS para personalizar los campos de entrada del formulario, incluyendo colores, espaciado y comportamiento del foco (`focus`).        |

### Ejemplo de uso

```astro
<FormularioContacto
  titleStyle="text-white font-bold text-3xl mb-6 text-center"
  titleFormulario="Contactanos"
  styleFormulario="bg-gradient-to-r from-[#05ce9135] to-indigo-600/35 backdrop-blur-sm p-2 w-96"
  colorLabel="text-white"
  styleInput="bg-white text-black focus:outline-none px-3 py-2 "
>
  <Button
    nameButton="Enviar"
    colorText="text-white"
    colorBackground="bg-green-600"
    colorBorder="border-blue-600"
    styleButton="w-full"
    onClick="alert('mensaje enviado');"
  />
</FormularioContacto>
```
